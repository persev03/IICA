# Datos semilla

Los únicos datos semilla admitidos en el repositorio son catálogos
administrativos estables, como códigos de país, moneda y nombres territoriales.

No se incluyen impuestos, incentivos, devolución de IVA, restricciones de
movilidad ni infraestructura como valores de prueba: esas reglas pueden cambiar
y deben cargarse desde una fuente oficial, con vigencia y URL de procedencia,
antes de incorporarse a una recomendación IICA.
